# Projection and Window Design

Load this file when the user needs help choosing projections or disentanglement windows.

## Projection guidance

- choose orbitals that match the chemistry and the target band character
- use the smallest physically complete basis that can represent the intended low-energy model
- if the target bands mix strongly across the energy window, explain why the projection set may need to expand

## Window guidance

- use disentanglement windows only when the manifold is truly entangled
- keep the frozen window tied to the subspace that must be represented accurately
- if the user cannot justify a window choice, say so and recommend checking the parent band structure first
